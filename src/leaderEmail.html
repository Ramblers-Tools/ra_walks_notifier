Leader Email Settings

The API token is used to look up the walk leader's email address from RA Mailman. If more than one matching profile is found, the app will skip the leader email rather than guess. Leave this blank when saving to keep the currently saved token unchanged.
While test mode is on, leader emails will only be sent if the resolved leader address matches one of these. Everyone else is skipped. Off by default so real leaders receive emails once you're ready.
