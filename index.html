Install instructions

  1. Download the latest DMG from GitHub.
  2. Open the DMG and drag Walks Manager Watch into Applications.
  3. Launch the app from Applications and complete setup.

What it does

A small desktop tool built for group walk approvals.

Checks Walks Manager

Runs scheduled checks against the configured Walks Manager group and keeps the pending count visible in the menu bar.

Sends clear emails

Notifies configured recipients when new or changed walks need attention, with HTML email summaries and direct links.

Handles setup locally

Stores SMTP settings, recipients, branding, group selection, and the Walks Manager browser session on the Mac.

Setup flow

The first launch collects the details needed for background checks and email notifications.

Add recipients

Enter one or more email addresses for walk review notifications.

Configure SMTP

Add host, port, username, password, from address, and display name.

Sign in

Open the Walks Manager login window, sign in, and let the app detect available groups.

Select group

Choose the group to monitor, then save setup and run the first check.
